# stdlinux dotfiles

## Terminfo
* Compile terminfo: `tic -o ~/.terminfo kitty.terminfo`

## Neovim
* Download a release
* Symlink the binary to `~/.local/bin`
* Install `vim-plug`

## zsh
* Install `zplug`
* Run `zplug install`
* Install `Pure` prompt manually
  * Do symlinks in `~/.zfunctions`

## ripgrep
* Download a release
* Symlink the binary to `~/.local/bin`
